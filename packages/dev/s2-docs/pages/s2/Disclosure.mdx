## Content

Disclosure follows the [ARIA Disclosure pattern](https://www.w3.org/WAI/ARIA/apg/patterns/disclosure/) and consists of two children: `DisclosureTitle` and `DisclosurePanel`. The `DisclosureTitle` controls the expansion of the `DisclosurePanel` and the contents inside.

```tsx render hideImports
"use client";
import {Disclosure, DisclosureTitle, DisclosurePanel} from '@react-spectrum/s2';

<Disclosure>
  <DisclosureTitle>Installation</DisclosureTitle>
  <DisclosurePanel>
    Install the package using npm or yarn: npm install @react-spectrum/s2
  </DisclosurePanel>
</Disclosure>
```

## Header customization

Use `DisclosureHeader` to add additional elements alongside the title, such as action buttons or icons.

```tsx render hideImports
"use client";
import {Disclosure, DisclosureTitle, DisclosurePanel, DisclosureHeader, ActionButton} from '@react-spectrum/s2';

<Disclosure>
  <DisclosureHeader>
    <DisclosureTitle>Project Settings</DisclosureTitle>
    <ActionButton>
      Edit
    </ActionButton>
  </DisclosureHeader>
  <DisclosurePanel>
    Configure your project settings including name, description, and permissions.
  </DisclosurePanel>
</Disclosure>
```

## Expansion state

Disclosure can be controlled or uncontrolled. Use `defaultExpanded` for uncontrolled behavior, or `isExpanded` with `onExpandedChange` for controlled behavior.

```tsx render hideImports
"use client";
import {Disclosure, DisclosureTitle, DisclosurePanel} from '@react-spectrum/s2';
import {useState} from 'react';

function Example() {
  let [isExpanded, setExpanded] = useState(false);

  return (
    <div style={{display: 'flex', flexDirection: 'column', gap: 16}}>
      <Disclosure defaultExpanded>
        <DisclosureTitle>Auto-expanded (Uncontrolled)</DisclosureTitle>
        <DisclosurePanel>This disclosure starts expanded by default.</DisclosurePanel>
      </Disclosure>
      
      <Disclosure isExpanded={isExpanded} onExpandedChange={setExpanded}>
        <DisclosureTitle>Controlled Expansion</DisclosureTitle>
        <DisclosurePanel>This disclosure is controlled by state.</DisclosurePanel>
      </Disclosure>
      
      <div>Controlled disclosure is: {isExpanded ? 'expanded' : 'collapsed'}</div>
    </div>
  );
}
```

## Events

Disclosure accepts an `onExpandedChange` prop which is triggered when it is expanded or collapsed.

```tsx render hideImports
"use client";
import {Disclosure, DisclosureTitle, DisclosurePanel} from '@react-spectrum/s2';
import {useState} from 'react';

function Example() {
  let [expandCount, setExpandCount] = useState(0);

  return (
    <div style={{display: 'flex', flexDirection: 'column', gap: 16}}>
      <Disclosure onExpandedChange={() => setExpandCount(count => count + 1)}>
        <DisclosureTitle>FAQ Item</DisclosureTitle>
        <DisclosurePanel>
          This is the answer to the frequently asked question.
        </DisclosurePanel>
      </Disclosure>
      <div>Expanded {expandCount} times</div>
    </div>
  );
}
```
