## Content

LinkButtons must have a visible label, and can optionally have an icon. Text only LinkButtons accept a string
as children. Icons can also be added as children, with a sibling `<Text>` element for the label.

```tsx render expanded
"use client";
import {LinkButton, Text} from '@react-spectrum/s2';
import New from '@react-spectrum/s2/icons/New';

<LinkButton href="https://react-spectrum.adobe.com/" target="_blank">
  <New />
  <Text>Open Link</Text>
</LinkButton>
```

## Events

LinkButtons support standard anchor events such as `onPress` for handling navigation programmatically.

```tsx render expanded
"use client";
import {LinkButton} from '@react-spectrum/s2';
import {useState} from 'react';

function Example() {
  let [lastClicked, setLastClicked] = useState('');

  return (
    <div style={{display: 'flex', flexDirection: 'column', gap: '12px'}}>
      <LinkButton 
        href="https://react-spectrum.adobe.com/" 
        target="_blank"
        onPress={() => setLastClicked('Link clicked!')}>
        Click me
      </LinkButton>
      <div>{lastClicked}</div>
    </div>
  );
}

<Example />
```
