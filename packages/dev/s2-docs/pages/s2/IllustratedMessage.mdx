## Content

An IllustratedMessage consists of three areas: an illustration, a title, and a body.
Each of these sections can be populated by providing the following components to your IllustratedMessage as children:
`svg`, Heading (title), and Content (body).

```tsx render expanded
"use client";
import {IllustratedMessage, Heading, Content} from '@react-spectrum/s2';
import Image from '@react-spectrum/s2/illustrations/gradient/generic1/Image';

<IllustratedMessage>
  <Image />
  <Heading>Drag and Drop your file</Heading>
  <Content>Select a File from your computer<br /> or Search Adobe Stock</Content>
</IllustratedMessage>
```

### With actions

IllustratedMessages can include action buttons to help users resolve the situation.

```tsx render expanded
"use client";
import {IllustratedMessage, Heading, Content, Button, ButtonGroup} from '@react-spectrum/s2';
import Image from '@react-spectrum/s2/illustrations/gradient/generic1/Image';

<IllustratedMessage>
  <Image />
  <Heading>No matching results</Heading>
  <Content>Try another search or browse our categories.</Content>
  <ButtonGroup>
    <Button variant="primary">Browse categories</Button>
    <Button variant="secondary">Clear search</Button>
  </ButtonGroup>
</IllustratedMessage>
```

### Accessibility

When an IllustratedMessage has an `svg` and a `Heading`, the `svg` needs an `aria-hidden` prop and a `role` prop set to `presentation`.

```tsx render
"use client";
import {IllustratedMessage, Heading, Content} from '@react-spectrum/s2';
import Image from '@react-spectrum/s2/illustrations/gradient/generic1/Image';

<IllustratedMessage>
  <Image aria-hidden role="presentation" />
  <Heading>No results found</Heading>
  <Content>Check your search terms and try again.</Content>
</IllustratedMessage>
```

## Labeling

### Accessibility

The title of an IllustratedMessage is typically provided via its `Heading`.
If a `Heading` isn't specified, the illustration needs an `aria-label` prop to identify it to assistive technology.
If you pass an SVG in yourself, you'll need to ensure that the `role` prop is set to `img` as well.

```tsx render expanded
"use client";
import {IllustratedMessage} from '@react-spectrum/s2';
import Image from '@react-spectrum/s2/illustrations/gradient/generic1/Image';

<IllustratedMessage>
  <Image aria-label="No results" />
</IllustratedMessage>
```
